# FiOWIN Lab 管理员编辑后台 - 快速开始指南

## 概述

FiOWIN Lab网站现已集成管理员可视化编辑后台，支持所见即所得的富文本编辑。您可以直接在浏览器中编辑网站内容，无需修改代码或重新部署。

## 快速开始

### 第一步：访问登录页面

打开浏览器，访问以下URL：
```
/admin
```

或使用完整URL：
```
https://3000-i4q1vkr1wysee0bon4nyh-891a9cd1.sg1.manus.computer/admin
```

### 第二步：输入登录凭证

在登录页面输入以下信息：

- **用户名**：`admin`（默认值，可在环境变量中修改）
- **密码**：`admin123`（默认值，可在环境变量中修改）

点击"登录"按钮。

### 第三步：进入编辑器

登录成功后，您将自动跳转到编辑器页面：
```
/admin/editor
```

## 编辑器功能

### 选择要编辑的页面

在编辑器顶部的下拉菜单中选择要编辑的页面：

| 页面 | 说明 |
|------|------|
| 关于实验室 | 实验室简介和基本信息 |
| 研究方向简介 | 研究方向的详细介绍 |
| 联系我们 | 联系方式和地址信息 |

### 编辑页面内容

#### 页面标题
在"页面标题"输入框中修改页面的标题。

#### 编辑内容区域
在下方的编辑区域中编辑页面内容。编辑器支持以下功能：

**文本格式**
- **粗体**：选中文本后点击粗体按钮，或使用快捷键 `Ctrl+B`
- **斜体**：选中文本后点击斜体按钮，或使用快捷键 `Ctrl+I`

**列表**
- **项目符号列表**：点击列表按钮创建无序列表

**链接**
- 点击链接按钮，输入URL地址，将选中文本转换为链接

**图片**
- 点击图片按钮，输入图片的完整URL地址（需要是公网可访问的URL）

### 保存内容

完成编辑后，点击页面底部的"保存页面"按钮。

**重要**：内容保存后会立即在网站前台生效，无需等待部署或刷新。

## 修改登录凭证

如果您需要修改管理员的用户名或密码：

1. 在Manus Management UI中打开项目设置
2. 进入 **Settings → Secrets** 面板
3. 找到以下环境变量：
   - `ADMIN_USERNAME`：管理员用户名
   - `ADMIN_PASSWORD`：管理员密码
4. 修改对应的值
5. 保存更改

修改后，使用新的凭证登录编辑器。

## 常见问题

### Q: 编辑后内容没有显示在网站上？
A: 请确保您点击了"保存页面"按钮。保存后内容应该立即生效。如果仍未显示，请刷新浏览器页面。

### Q: 如何添加新的可编辑页面？
A: 目前支持编辑的页面已预设。如需添加新页面，请联系技术人员修改编辑器配置。

### Q: 图片无法上传怎么办？
A: 编辑器不支持直接上传图片，需要提供图片的完整URL地址。您可以：
1. 将图片上传到云存储服务（如阿里云OSS、腾讯云COS等）
2. 获取图片的公网访问URL
3. 在编辑器中粘贴该URL

### Q: 忘记了登录密码怎么办？
A: 请在Manus Management UI的 Settings → Secrets 面板中重置 `ADMIN_PASSWORD` 环境变量。

## 编辑示例

### 添加标题和段落

1. 在编辑区域中输入文本
2. 选中标题文本，点击粗体按钮使其加粗
3. 按Enter键创建新段落

### 创建列表

1. 点击列表按钮
2. 输入列表项内容
3. 按Enter键添加新的列表项
4. 按Backspace键删除列表项或退出列表

### 添加链接

1. 选中要转换为链接的文本
2. 点击链接按钮
3. 输入目标URL（例如：https://example.com）
4. 点击确认

### 插入图片

1. 点击图片按钮
2. 输入图片的完整URL（例如：https://example.com/image.jpg）
3. 点击确认

## 技术信息

- **编辑器**：TipTap（基于Prosemirror的现代富文本编辑器）
- **存储**：MySQL数据库
- **API**：tRPC（类型安全的RPC框架）
- **前端框架**：React 19 + Tailwind CSS
